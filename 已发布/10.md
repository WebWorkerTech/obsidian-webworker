Web Worker 播客已经做到第十期了，这次我们邀请了一个老朋友作为新的嘉宾刘威 Franky。同时我们还邀请了刘威的老朋友 小白菜 作为面试官（不是）。我们采访一下刘威是如何进行前端程序员的个人成长、系统学习的。同时我们也对 Web Worker 的未来样子做了展望和吹牛逼。来听听看第一季的最后一集 Web Worker 播客有什么有意思的内容吧！
